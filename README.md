# STM32 Proximity Sensor

A simple proximity and distance measurement system built using an STM32 microcontroller and an ultrasonic sensor. The project measures distance in real time and prints the readings over UART.

---

## Features
- Ultrasonic-based distance measurement  
- STM32 HAL-based implementation  
- Timer Input Capture for accurate pulse width timing  
- UART output for monitoring readings  

---

## Hardware Used
- STM32 Development Board  
- HC-SR04 Ultrasonic Sensor  
- Jumper wires
